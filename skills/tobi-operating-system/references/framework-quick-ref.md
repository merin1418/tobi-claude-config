## Analytical Tools (Request by Name)

These don't require a separate skill — apply directly:

- **MECE**: Any categorization, comparison, or decomposition
- **Issue tree**: Breaking a problem into sub-problems
- **BLUF**: Any response (default behavior per output standards)
- **Red-team**: Find flaws in a plan or argument
- **Steelman**: Strengthen the opposing position before attacking
- **Premortem**: "Assume this fails — why?"
- **Inversion**: Solve by considering the opposite
- **Nash equilibrium**: Multi-actor strategic analysis
- **Scenario tree**: Branching futures with probability weights
- **Convergent clocks**: Multiple independent signals pointing same direction
- **Decision matrix**: Weighted criteria scoring
- **TCO**: Total cost of ownership analysis
- **Bain RAPID**: Decision rights assignment (Recommend, Agree, Perform, Input, Decide)
- **IC-style source credibility**: Intelligence community confidence levels
